Installation
============

Execute these commands in command line:

	cd ~
	mv .vim vim_backedup
	mv .vimrc vimrc_backedup
	git clone https://github.com/shahverdy/.vim
	mv .vim/.vimrc .vimrc
	~/.vim/install.sh
	vim -c 'PluginInstall'
